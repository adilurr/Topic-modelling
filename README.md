# Topic Modeling and Document Categorization

## Overview

This project focuses on Natural Language Processing (NLP) techniques to discover hidden topics within large collections of text documents and automatically categorize them.

The project uses topic modeling approaches such as Latent Dirichlet Allocation (LDA) and Non-Negative Matrix Factorization (NMF) to analyze unstructured textual data.

## Objectives

- Extract meaningful topics from documents
- Organize large text datasets
- Improve document understanding
- Automate text categorization

## Features

- Text preprocessing
- Stopword removal
- Tokenization
- Topic extraction using LDA
- Topic extraction using NMF
- Document categorization
- Data visualization

## Technologies Used

- Python
- Scikit-learn
- Pandas
- NumPy
- NLTK
- Matplotlib

## Workflow

1. Load text dataset.
2. Clean and preprocess data.
3. Convert text into numerical features.
4. Apply topic modeling algorithms.
5. Extract and visualize topics.
6. Categorize documents based on topic distribution.

## Results

The project successfully identifies hidden themes within large document collections and helps organize unstructured text into meaningful categories.
